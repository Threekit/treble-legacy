## Player

The `<Player />` component renders out the Threekit Player.

```jsx
import React from 'react'
import ReactDOM from 'react-dom'

import { Player } from 'threekit'

const Component = () => {
  return (
    <div>
      <Player />
    </div>
  )
}
```
