# Product Name

### Display the Product's Name

The `<Title />` component will display the title of the Product used to
initialize the Threekit Player.

It's also possible to display a custom title by passing a string into the `title` prop.

## Demo

    name="Default"
    args={{
      title: 'Product Title',
    }}

  </Story>
