# Zoom

### A Widget for changing the Zoom of the active Threekit Player Camera

## Overview

The Zoom widgets offers a Click interaction based alternative to the existing
zoom feature that comes with the Threekit Player.

The component comes in both a **horizontal** and **vertical** orientation for placement
on any side of the player.
