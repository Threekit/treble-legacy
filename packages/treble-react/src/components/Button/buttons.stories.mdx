## Overview

The `<Button />` component can be used anywhere in the app.
It does not connect to any attribute type or even to the
ThreekitProvider and as such it can be used as a general
UI element.

## How to Use

Example of how to use a **Button** component:

`<Button title="Button Title" onClick={() => console.log('clicked')} />`
