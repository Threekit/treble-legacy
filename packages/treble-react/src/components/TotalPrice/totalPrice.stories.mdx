# Price

### Display the Configuration's Total Price

This component will display the total price of the configuration.

Passing a string/number into the `price` prop will display that
value instead.

  <Story
    name="Default"
    args={{
      price: '$500',
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>
