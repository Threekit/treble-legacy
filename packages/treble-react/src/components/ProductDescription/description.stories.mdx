# Description

### Display a Product's Description

The `<Description />` component will display a description for the product initialized
in the Threekit Player. By default, the description data comes
from the metadata on the initialized Catalog Item, under the key
`_description`.

It's also possible to display a custom description by passing a string into the `description` prop.

<Canvas>
  <Story
    name="Default"
    args={{
      description: 'This is the description for a single-product configurator.',
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>
