# Treble Framework

## Getting Starting

### Cloning the repository

```bash
git clone git@github.com:Threekit/treble.git
cd ./treble
```

### Scripts

### Installing dependencies

```bash
npm install

```

#### Start local dev server

```bash
npm run dev
```

#### Lint check all packages

```bash
npm run lint
```

#### Format all packages

```bash
npm run format
```
